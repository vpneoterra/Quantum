B
Barren plateau
A phenomenon in variational quantum algorithms where the gradient of the cost function vanishes exponentially with system size, making optimization intractable.
Below-threshold operation
Operating a quantum error-correcting code at physical error rates below the code’s fault-tolerance threshold, enabling logical error suppression with increasing code distance.
Bivariate bicycle code
A specific qLDPC code (e.g., [[144,12,12]]) that encodes multiple logical qubits per block with significantly lower overhead than surface codes.
Break-even point
The threshold where a logical qubit encoded with quantum error correction outperforms its constituent physical qubits in error rate.
C
Cat qubit
A qubit encoded in superpositions of coherent states (α and −α) in a microwave cavity; exponentially suppresses bit-flip errors.
Circuit depth
The number of sequential gate layers in a quantum circuit; deeper circuits require longer coherence times.
Circuit knitting
A technique that decomposes a large quantum circuit into smaller subcircuits executable on available hardware, recombining results classically.
CLOPS
Circuit Layer Operations Per Second; a throughput benchmark measuring how fast a quantum computer executes parameterized circuit layers.
Coaxmon
A 3D superconducting qubit architecture used by Oxford Quantum Circuits (OQC) enabling scalable connectivity.
Code distance
The minimum number of physical qubit errors needed to cause an undetectable logical error; higher distance provides greater protection.
Coherence time (T1)
Energy relaxation time; how long a qubit retains its excited state before decaying to the ground state.
Coherence time (T2)
Dephasing time; how long a qubit maintains phase coherence between superposition states.
Color code
A topological QEC code on a 3-colorable lattice; supports transversal Clifford gates (H, S, CNOT).
Cross-entropy benchmarking (XEB)
Google’s method to verify quantum circuit fidelity by comparing output distributions to ideal classical simulations.
D
Decoding
Classical computation that interprets syndrome measurements and determines the correction to apply to an error-corrected quantum state.
Dilution refrigerator
A cryogenic cooling system reaching approximately 10–15 millikelvin; required to operate superconducting and some spin qubits.
E
Eastin-Knill theorem
The proof that no quantum error-correcting code can implement a universal gate set entirely through transversal gates.
Encoding rate
The ratio of logical qubits to total physical qubits in an error-correcting code; surface code has rate approximately 1/d².
Error suppression factor (Λ)
The ratio by which logical error rates decrease when code distance is increased by one step; Λ > 1 indicates below-threshold operation.
Error threshold
The maximum physical error rate below which quantum error correction can suppress logical errors to arbitrarily low levels.
F
Fault-tolerant quantum computing (FTQC)
Quantum computation performed using error-corrected logical qubits such that errors can be suppressed to arbitrarily low rates during long computations.
Fluxonium
A superconducting qubit with a superinductance shunting the Josephson junction; exhibits very long T1 but slower gate speeds.
Fusion gate
A probabilistic two-qubit gate in photonic quantum computing that fuses two resource states; basis of PsiQuantum and Xanadu approaches.
G
Gate fidelity
The probability that a quantum gate operation produces the correct output; typically reported as a percentage (e.g., 99.9%).
GKP state
A bosonic code state (Gottesman-Kitaev-Preskill) that encodes a qubit in the position/momentum of a harmonic oscillator.
H
Heavy-hex topology
IBM’s qubit connectivity layout with degree-2 and degree-3 nodes arranged in hexagonal patterns.
I
Ion trap (Paul trap)
A device confining charged atoms using oscillating electric fields; the basis for trapped-ion quantum computers.
L
Lattice surgery
A technique to perform logical gates between surface-code patches by merging and splitting code boundaries.
Logical qubit
A qubit encoded redundantly across multiple physical qubits using an error-correcting code, enabling fault-tolerant operations.
M
Magic state distillation
A protocol to produce high-fidelity non-Clifford (T-gate) resource states from many noisy copies; required for universal fault-tolerant computation.
Majorana zero mode
A non-Abelian anyon predicted to form at endpoints of topological superconductors; the theoretical basis for topological qubits.
Mosca’s Theorem
A risk framework: if data sensitivity lifetime (X) plus migration time (Y) exceeds time until a cryptanalytically relevant quantum computer (Z), the data is at risk now.
O
Optical tweezer array
A lattice of tightly focused laser beams used to trap and individually address neutral atoms for quantum computation.
P
Physical qubit
A single hardware-level quantum bit; subject to noise, decoherence, and gate errors.
Q
Q-Day
The hypothetical date when a quantum computer can break current public-key cryptographic standards (specifically RSA-2048) within a practical timeframe.
Quantum advantage
Demonstration that a quantum computer solves a practical, useful problem faster than any known classical method.
Quantum annealing
A heuristic optimization method using quantum tunneling to find low-energy states of a cost function.
Quantum error detection
Identifying that an error has occurred (without necessarily correcting it); requires fewer resources than full quantum error correction.
Quantum interconnect
A link (optical fiber, microwave, etc.) that transfers quantum states between separate QPU modules.
Quantum supremacy
A computational task performed by a quantum computer that no classical computer can achieve in feasible time.
Quantum teleportation
Transfer of quantum information using entanglement and classical communication; used for inter-module operations.
Quantum volume
A single-number benchmark (2^n) capturing the largest square (n×n) random circuit a QPU can execute with >2/3 success probability.
R
Randomized benchmarking
A protocol to estimate average gate error rates by applying random sequences of Clifford gates.
Repetition code
The simplest QEC code; protects against one error type by repeating a qubit across multiple physical qubits.
Rydberg interaction
Strong, tunable interaction between neutral atoms excited to high-energy Rydberg states; enables two-qubit entangling gates.
S
Surface code
A topological QEC code using a 2D grid of qubits with nearest-neighbor interactions; threshold approximately 0.5–1.0%.
Syndrome measurement
Measuring ancilla qubits to extract error information without disturbing the encoded logical state.
T
Tensor network
A classical simulation technique that represents quantum states as networks of contracted tensors; effective for low-entanglement circuits.
Topological gap protocol
Microsoft’s experimental method for verifying topological properties in Majorana devices; subject to critique regarding false positives.
Topological protection
Error resilience derived from encoding quantum information in global, topological properties of a system rather than local degrees of freedom.
Transmon
A charge-insensitive superconducting qubit (variant of the Cooper-pair box) with E_J >> E_C; the dominant design used by IBM, Google, and Rigetti.
Transversal gate
A logical gate implemented by applying independent physical gates to each qubit in a code block; inherently fault-tolerant.
