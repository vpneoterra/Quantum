Glossary & Terms
ASIC
Application-Specific Integrated Circuit — custom chip for a single workload (e.g., inference). Google TPU, AWS Inferentia, Cerebras WSE.
Blackwell
NVIDIA GPU architecture (B200, B300). FP4 precision, up to 288GB HBM3e, 20 PFLOPS.
CDNA
Compute DNA — AMD's data center GPU architecture. CDNA 3 = MI300X, CDNA 4 = MI355X.
CPU
Central Processing Unit — general-purpose processor. ~29% of inference compute share.
CUDA
Compute Unified Device Architecture — NVIDIA's parallel computing platform. 4M+ developers.
FLOPS
Floating Point Operations Per Second. PFLOPS = 10¹⁵ FLOPS.
FP4 / FP8 / FP16
Floating-point precision formats. Lower precision = less memory and compute per token.
FPGA
Field-Programmable Gate Array — reconfigurable chip with deterministic latency. ~4% share.
GPU
Graphics Processing Unit — massively parallel processor. 52.1% of inference compute.
Hopper
NVIDIA GPU architecture (H100, H200). H100 set FP8 standard; H200 added 141GB HBM3e.
LPU
Language Processing Unit — Groq's custom chip for deterministic low-latency inference.
NPU
Neural Processing Unit — dedicated AI accelerator in phones/PCs/edge. 970M shipped 2025.
PFLOPS
PetaFLOPS — 10¹⁵ floating-point ops/sec. Chip and data-center performance measure.
ROCm
Radeon Open Compute — AMD's open-source GPU platform. Primary CUDA competitor.
SoC
System on Chip — CPU + GPU + NPU on a single die. Smartphones and edge devices.
TOPS
Tera Operations Per Second — NPU/accelerator performance measure for on-device AI.
TPU
Tensor Processing Unit — Google's custom ASIC. 7 generations (v1–v7 Ironwood).
WSE
Wafer-Scale Engine — Cerebras full-wafer chip (4T transistors, 900K cores).
CoWoS
Chip-on-Wafer-on-Substrate — TSMC's 2.5D packaging bonding GPU to HBM. Tightest supply bottleneck.
DDR
Double Data Rate — standard memory (DDR4/DDR5). Lower bandwidth than HBM, significantly cheaper.
HBM
High Bandwidth Memory — stacked DRAM for extreme throughput. Binding constraint for LLM inference. HBM2e → HBM3 → HBM3e → HBM4.
KV-Cache
Key-Value Cache — memory structure storing attention states during autoregressive LLM generation.
MR-MUF
Mass Reflow Molded Underfill — SK Hynix's HBM packaging technology. Higher yields, better thermals.
TSMC
Taiwan Semiconductor Mfg Co — fabricates 90%+ of leading-edge AI chips worldwide.
AI
Artificial Intelligence — simulation of human intelligence by computer systems.
Edge AI
Inference at or near the data source. Driven by latency, privacy, and bandwidth needs.
GenAI
Generative AI — models producing new content (text, images, code, audio).
Inference
Running a trained model to generate predictions/outputs on new data. Focus of this report.
Inference Flip
Late-2025 crossover where inference compute/revenue surpassed training. Structural shift.
LLM
Large Language Model — billions of parameters trained on text. GPT-4, Llama, Claude, Gemini.
MLOps
Machine Learning Operations — deploying, monitoring, maintaining ML models in production.
MoE
Mixture-of-Experts — activates subset of parameters per token, reducing compute per inference.
Quantization
Reducing precision (FP16 → FP4) to cut memory/compute with minimal accuracy loss.
RAG
Retrieval Augmented Generation — LLM + external knowledge retrieval architecture.
Speculative Decoding
Small draft model predicts tokens verified in parallel by the large model. Reduces latency.
Token
Fundamental text unit for LLMs (word or fragment). Pricing is per input/output token.
BESS
Battery Energy Storage System — on-site batteries for backup power and peak-shaving.
GW
Gigawatt — one billion watts. Measures data center power capacity and grid supply.
NIC
Network Interface Card — connects servers to network. AI inference NICs are fastest-growing segment.
NIM
NVIDIA Inference Microservices — containerized endpoints optimized for NVIDIA hardware.
OCP
Open Compute Project — open-source data center hardware specifications initiative.
SOFC
Solid Oxide Fuel Cell — behind-the-meter power generation for grid-independent edge DCs.
Sovereign AI
National AI infrastructure for domestic compute and data sovereignty. 50+ countries.
TWh
Terawatt-hour — one trillion watt-hours. Annual data center electricity consumption.
ASP
Average Selling Price — mean price per unit across a product category.
CAGR
Compound Annual Growth Rate — annualized average growth rate over a period.
CapEx
Capital Expenditure — funds for acquiring/maintaining physical infrastructure.
OpEx
Operating Expenditure — ongoing costs: electricity, cooling, maintenance.
TAM
Total Addressable Market — total revenue at 100% market share.

The Inference Flip

In late 2025, inference surpassed training as the majority of data center AI revenue.

Structural, not cyclical. A model is trained once but serves billions of inference requests. Inference grows from 40% to 80% of all AI compute by 2030.

Training is bursty, centralized, CapEx-heavy. Inference is continuous, distributed, OpEx-focused. The industry enters the "Utility Era" — inference consumed like electricity. ChatGPT: 800M+ weekly users.
